# Genacvid's Godot Project Template (for Godot 4.6)
This is a template for creating 3D singleplayer/multiplayer games.
It contains a bunch of useful entity components, state machines, and singletons.

## State Machines
	Movement
	Behavior

## Entities
	Player
	TestMob
	Weapon
	Explosion
	Projectile

## Components
	Attack
	Damage
	Input
	Interact
	Inventory
	Move
	Navigation
	Persist
	Sight
	Trace
	VisualModifier

## Level FGD (TrenchBroom support provided by FuncGodot)
	Navigation Hints
	Movers
	Pushers
	Rotaters
	Buttons
	Doors
	Water

## Singletons
	Game
	Save
	Setting
	Loader
	Music
	Debug (provided by Debug Draw 3D)

## Networking
	ENetMultiplayerPeer
	SteamMultiplayerPeer (provided by GodotSteam plugin)
